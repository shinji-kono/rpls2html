# rpls2html

# requirements
   nkf : https://github.com/nurse/nkf
   NKF.pm : Perl module from above
and
   Perl

# usage

rpls.pl : decode *.rpls and genetate html
   -g  debug mode
   -d  show detail
   -s  simple mode, no html tag
   -r  recursive mode via find -s
   -h  show tnis
